2. CSS Files:
  I'm using some CSS files in this template.	
  The style.css file contains all of the specific stylings for the page. The file is separated into sections using:

  /* Porfolio
  ------------------------------------------*/
  some code

  /* Blog
  ------------------------------------------*/	
  some code

  /* Services
  ------------------------------------------*/
  some code

  etc, etc. 

  If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.
  The rest of css files are using javascript of it's plugin such as superslides.css used for javascript jquery.superslides.min.js, lightbox.css used for javascript lightbox-2.6.min.js.
